# Ship It

**The Self-Taught Developer's Guide to Building Products and Starting Up**

> From first idea to first user — no MBA required.

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

A free, open-source book for self-taught developers who want to stop just coding and start building real products and startups.

This book is written for someone who knows how to code but doesn't know how to turn that skill into a product or startup. You'll learn the full journey: finding real problems, designing solutions, building MVPs, deploying, getting users, iterating based on feedback, and building in public.

**Read online**: [ship-it-book.github.io](https://ship-it-book.github.io)

## 📖 How to Read

**Online**: Visit [ship-it-book.github.io](https://ship-it-book.github.io)

**Locally**: Clone the repo and run the Docusaurus dev server:

```bash
git clone https://github.com/ship-it-book/ship-it-book.git
cd ship-it-book
npm install
npm start
```

**Offline**: Read the markdown files directly in the `docs/` directory.
